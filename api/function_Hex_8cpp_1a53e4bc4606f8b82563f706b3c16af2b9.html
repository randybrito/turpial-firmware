esp_err_t util::bytesToHex(std::uint8_t *buf, char *dst, std::size_t len)¶

Converts an array of bytes to a hexadecimal string.

Return

  • ESP_OK: succeed

  • ESP_FAIL: an invalid character was found

Parameters

  • buf: null terminated hexadecimal source string

  • len: byte size

  • [out] dst: hex array
